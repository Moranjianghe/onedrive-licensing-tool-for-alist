AList OneDrive 授權工具

關於此工具

此工具幫助您獲取 OneDrive 的授權認證，便於在 AList 中掛載您的 OneDrive 儲存空間。

使用此工具，您將獲得 AList 掛載 OneDrive 所需的 refresh_token 等授權憑證。

使用方法

  1. 點擊下方「取得 OneDrive 授權」按鈕 (強烈推薦使用「直接 OAuth 流程」)
  2. 登入您的 Microsoft 帳戶
  3. 授予必要的權限
  4. 系統會自動獲取並顯示需要的授權信息

解決常見問題

遇到 "AADSTS9002313: Invalid request" 錯誤?

這通常是因為在 AList 中使用的令牌格式不正確。請使用「直接 OAuth 流程」獲取令牌，它提供的 refresh token 可以正確用於 AList。

一般來說，使用 Microsoft 的 MSAL 庫會限制對 refresh token 的訪問，這是為什麼我們提供兩種授權方式的原因。

選擇授權方式

推薦方式： 使用直接 OAuth 流程可獲取 refresh token（長期有效，最適合 AList）

替代方式： 使用 MSAL 庫流程可能只獲取 access token（短期有效，可能需要定期更新）

推薦方式：使用直接 OAuth 流程 替代方式：使用 MSAL 庫流程

此工具僅用於獲取合法授權，請勿用於非法用途。
